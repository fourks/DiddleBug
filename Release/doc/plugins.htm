Description

IntelliBoogerTM plugins are separate palm programs that you can install to extend the capabilities of DiddleBug. Plugins will not show up in your Launcher application, but will be available from the plugin button in DiddleBug after you select "Transfer" from the menu. You can delete plugins using your file-manager or the built-in function in the PalmOS launcher. Each plugin knows how to write to the database of a given application and how to jump to that application. You can install as many or as few plugins as you want, so you do not have to waste your RAM on unwanted capabilities. Only the plugins that you install will show up in your plugin menu.

A list of the currently available plugins, and a description of their capabilities is given below. Plugins that are planned in the future are also available. You should also read the section in the manual about using the plugins.

Plugin Descriptions

A list of plugins and their capabilities is given here. All plugins are available in the zip-file from the Download area.

ToDo  (v.1.4, size: 2.9K)

The ToDo plugin will write the text you have entered in the pick field into the database of the built-in ToDo application. If you have an alarm set in DiddleBug, it will use the alarm date as the due date for the new entry.
 

DateBook  (v.1.3, size: 3.0K)

The DateBook plugin will write the text you have entered in the pick field as a new appointment in the built-in DateBook application. It will use any alarm or repeat information that you have set in DiddleBug for the new entry.

This is not a plugin for the "Calendar" application on newer palmOne devices (i.e. Tungsten T3). If you want a "Calender" plugin, please write to palmOne to release the necessary information for their new record format.
 

Address   (v.1.6, size: 3.1K)

The Address plugin will create a new contact in the built-in Address application. It will write the text you have entered in the pick field into the first phone entry if it starts with a number or a plus sign '+' (international dialing prefix). If you write the entry as:

name; first name; company name

it will put the data in the appropriate fields. You may leave any of the three fields blank, but you must have the two semi-colons ';' (e.g., 'Smith; ; Palm Computing'). If you don't use the preceeding format, and the entry contains a space, it will be put in the company field. Otherwise, for single words, it will go in the last name field.

This is not a plugin for the "Contacts" application on newer palmOne devices (i.e. Tungsten T3). If you want a "Contacts" plugin, please write to palmOne to release the necessary information for their new record format.

Memo Pad  (v.1.6, size: 2.3K)

The Memo Pad plugin will create a new memo with the text in the pick field (and the note attached to the sketch, if one exists) and jump to that memo if you specify GoTo.
 

ToDo (for DateBk5)   (v.1.0, size: 2.9K)

This plugin will create a new entry in the ToDo database using the text you have entered in the pick field. You have the option of jumping to the entry in DateBk5 using the GoTo option.
 

Event (for DateBk5)   (v.1.0, size: 3.1K)

This plugin will create a new event in DateBk5, and optionally jump to it. Any alarm or repeat information set in DiddleBug will be used in the new entry.
 

Floating Event (for DateBk5)   (v.1.0, size: 3.3K)

This plugin will create a new floating event in DateBk5, and optionally jump to it. Any alarm or repeat information set in DiddleBug will be used in the new entry. In addition, you can mark the new entry as complete from within DiddleBug using the flick menu.

Hi-Note (picture)   (v.1.5, size: 2.3K)

This is the first plugin that will actually transfer the sketch data in addition to the text. This plugin will create a new picture entry in Hi-Note and optionally jump to that entry. The text you enter in the pick field is used as the title for the new sketch. Thanks to Cyclos software for providing the information necessary to create this plugin.
 

Hi-Note (text)   (v.1.5, size: 1.9K)

This plugin will use the text you've entered in the pick field to create a new text item in Hi-Note and optionally jump to that entry.
 

Big ToDo   (v.1.2, size: 2.9K)

This plugin will create an entry in the "Big ToDo" application from Palm Planning. You may optionally jump to the application or set the complete flag from within DiddleBug. The day of the current DiddleBug alarm (if any) will be used as the due date for the new entry.
 

Diddle   (v.1.3, size: 1.8K)

This plugin will copy the DiddleBug sketch into the Diddle database. The text entered into the pick field will be used for the title of the sketch. Diddle can be used as a more permanent storage area for your sketches, and will also allow you to link to your sketches using Linker or LinkMaster.

This plugin does not work with hi-res sketches.
 

FireViewer   (v.1.0, size: 1.7K)

This plugin will create a new sketch in FireViewer and optionally jump to it. The text you enter in the pick field will be used for the title of the new picture. This is useful for exporting DiddleBug sketches to be manipulated or converted by the many tools available for FireViewer.
 

TinyViewer   (v.1.3, size: 1.7K)

This plugin will copy your current DiddleBug sketch into a standalone database of ImageViewer format. The text in the pick field will be used as the name of the database. You may optionally jump to TinyViewer to view your new database. Please note that you will always end up in the index view of TinyViewer, and not directly on the newly created sketch.
 

Spec   (v.1.3, size: 1.7K)

This plugin will copy your current DiddleBug sketch into a standalone database of ImageViewer format. The text in the pick field will be used as the name of the database. You may optionally jump to Spec to view your new database. Please note that you will not necessarily jump to your newly created image, and may have to page around to find it.
 

ClipBoard   (v.1.3, size: 1.5K)

This is a donated plugin by Lonnie Foster that copies text from the pick field into the clipboard. If you select "GoTo", then you will bring up your launcher application.
 

HandBase   (v.1.3, size: 1.5K)

This is an "always-goto" plugin that copies your text into the clipboard, then jumps to HandBase. From there you can paste your text into whichever database/record you want.
 

BrainForest   (v.1.3, size: 1.5K)

This plugin will copy the current pick text to the clipboard, then jump to the BrainForest application. You are expected to paste the text from the clipboard into Brainforest at the desired location.
 

JShopper   (v.1.4, size: 1.8K)

ToDo Plus   (v.1.5, size: 2.9K)

This plugin will create a new ToDo item using the text you enter into the pick field and jump to that item in ToDo Plus. If you have a DiddleBug alarm set, that date will be used as the due date for the ToDo, but no alarm or sketch data will be transferred.

NOTE: The version 1.0 of this plugin was mistakenly released with the wrong creator id, and will conflict with the ListMaker plugin. Please uninstall version 1.0 of this plugin before installing version 1.5.
